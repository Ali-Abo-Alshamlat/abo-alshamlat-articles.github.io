This is a writeup and first-time walkthrough of the 
            Burp Suite: The Basics room
            on the TryHackMe Cybersecurity training platform. The Attack box
            virtual machine was used to walk through the room task questions.


            The room groups and splits tasks into several sections:

            Introduction : Task 1
            Getting Started: Tasks 2–7
            Proxy: Tasks 8–13
            Practical : Task 14
            Conclusion: Task 15
