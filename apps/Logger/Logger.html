This is a beginner-level forensics challenge from HackTheBox, involves a
            document with USB Keylogger Payloads, and you must figure out what it is doing.

            

            First of all, I downloaded files from HackTheBox and after that, I unzipped
            the downloaded file. After unzipping I found one pcapng file in that so I
            opened that file in Wireshark and started analyzing that pcapng file.

            

            After Analyzing that I found some keystrokes by devices.

            

            So finally I found keystrokes in that file so now I have to grab only those
            packets. so applied some filters and finally, I got one filter to extract only
            original keystrokes from that file.

            

                Filter: usb.transfer_type == 0x01 and frame.len == 35 and 
                !(usb.capdata == 00:00:00:00:00:00:00:00)
            


            after applying the filter I exported those packets by going to File > 
            Export Specified Packets.

            I learned a much easier way when researching this write-up was to use
            tshark to extract the leftover capture data (be sure to pipe to tr -d: to get rid
            of the colons in the output. Redirect output to a text file, once you’ve
            confirmed it looks good.

            Command: tshark -r final.pcapng -T fields -e usb.capdata | tr -d :

            

            tshark -r final.pcapng -T fields -e usb.capdata | tr -d : >> keystroke.txt
