                Scanning and Enumeration
            


            

                Nmap
            


            As the room’s title suggested, let’s run Nmap :

            

                
                    
                        nmap -A -p- -vv -T4 IP_TARGET
                    
                
            


            

            

            

                Open ports
            


            Ports 22 and 2222 are ssh ports, let’s check the 31377 port.

            This port’s information returned by nmap seems really interesting, let’s analyze this.

            We can quickly see user and password information :

            

                
                    
                        user:pass
                        ubuntu:Dafdas!!/str0ng
                    
                
            


            

                Connecting with found credentials
            


            In the Nmap results, we have seen that SSH ports 22 and 2222 are open, 
            try to connect to the machine with our freshly found credentials :

            

                
                    
                        ssh ubuntu@IP_TARGET
                    
                
            


            Now we are connected !

            

                Get the Flag
            


            This is a CTF so let’s check in the directories around :

            

                
                    
                        ls -al
                    
                
            


            

            Nothing great. Let’s see if there are other users.


            

                
                    
                        cd ..
                        ls -al
                    
                
            


            

            Let’s see if there is something for us in the user directory :

            

                
                    
                        cd user
                        ls -al
                    
                
            


            And…that’s it, our file containing the flag to owned this room !

            

            This is a very easy room but it’s always a good thing to know 
            how to use the basic tools and how to efficiently begin a pentesting.
