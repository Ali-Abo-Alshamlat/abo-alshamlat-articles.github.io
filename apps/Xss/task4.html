            Ques 1: Which one of the following characters do you expect to be encoded?
            ., ,, ;, &, or #?
    
            

                Answer: &
            


            Which one of the following characters do you expect to be encoded? +, -,
            *, <, =, or ^?

            

                Answer: <
            


            Which function can we use in JavaScript to replace (unsafe) special
            characters with HTML entities?

            

                Answer: escapeHtml()
            


            Which function did we use in PHP to replace HTML special characters?

            

                Answer: htmlspecialchars()
